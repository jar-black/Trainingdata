# Verner von Heidenstam Poems Collection

This directory contains Verner von Heidenstam's poems in paired Swedish-English format.

## File Format

Each poem is stored as a pair of files:
- `poem1-swedish.txt` - Swedish original
- `poem1-english.txt` - English translation
- `poem2-swedish.txt` - Swedish original
- `poem2-english.txt` - English translation
- ...and so on

## About Verner von Heidenstam

Verner von Heidenstam (1859-1940) was a Swedish poet and novelist who received the Nobel Prize in Literature in 1916 "in recognition of his significance as the leading representative of a new era in Swedish literature."

## Goal

Collect 25-30+ poem pairs with both Swedish originals and English translations, focusing on Heidenstam's most significant works from his major collections.

## How to Use This Collection

1. Each poem has two files: one Swedish original, one English translation
2. Files are numbered sequentially (poem1, poem2, etc.)
3. Metadata is stored in `/home/user/Trainingdata/data/heidenstam_poems_template.jsonl`
4. Use the metadata file to track titles, collections, years, and sources

## Notes

- All translations are by Charles Wharton Stork unless otherwise noted
- Swedish texts are from Projekt Runeberg's digital editions
- Some poems may have alternative titles in different editions
- Heidenstam's poetry is characterized by vivid imagery, national romanticism, and classical influences
